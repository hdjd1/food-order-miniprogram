# 扫码点餐小程序 README.md
# food-order-miniprogram
食扫点单小程序｜餐厅扫码点餐系统
基于扣子编程开发的前端点餐小程序项目，顾客扫码即可在线选菜、下单，模拟线下餐饮门店点单场景。

## 📖 项目简介
本项目是一套面向餐饮门店的**扫码点单小程序原型**，实现顾客自助点餐流程：
- 菜品分类展示：招牌推荐、热菜、凉菜、主食
- 菜品详情、价格、销量展示
- 购物车增减菜品
- 订单查看功能
- 移动端适配，支持小程序预览

## ✨ 功能清单
- ✅ 菜品分类Tab切换
- ✅ 菜品图文展示、销量、价格渲染
- ✅ 加入购物车、数量增减
- ✅ 我的订单页面
- ✅ AI动态更新菜品图片（项目内特色功能）
- ✅ 移动端UI适配

## 🛠 开发环境
- 开发平台：扣子编程 code.coze.cn
- 项目类型：小程序前端项目
- 构建工具：pnpm

## 🚀 项目运行方式
### 方式1：扣子编程直接导入（推荐）
1. 将本仓库代码下载至本地
2. 打开【扣子编程】→新建项目
3. 上传全部代码文件
4. 安装依赖，启动预览，直接在右侧查看小程序效果

### 方式2：本地运行
```bash
# 安装依赖
pnpm install
# 启动项目
pnpm dev
```
